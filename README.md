# -Desafio_Criativo_Extraindo_Insights_do_Feedback_de_Clientes_Bancarios
Nesta atividade, são seguidas algumas etapas simples que, ao final, irão ajudar a extrair insights de feedbacks de clientes bancários com apoio de IA, usando contexto, critérios de análise e cuidados com dados sensíveis. Lembrando que dados inseridos aqui não existem no mundo real e foram gerados por uma IA.
